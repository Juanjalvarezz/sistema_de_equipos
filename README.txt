Pasos

	1.- Se debe agregar a la carpeta el archivo .env (Archivo proporcionado en la actividad)
	2.- Se debe correr el Xampp y agregar la base de datos (Archivo proporcionado en la actividad)
	3.- Se debe abrir la carpeta en el Visual Code
	4.- Se debe colocar en la consola el siguiente comando "npm run server"
	5.- Luego para Manejar las consultas se debe seguir los pasos del archivo "Informe Unidad III"
